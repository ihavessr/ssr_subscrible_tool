 ＃  SSR节点列表生成器
 >适应ios版shadowrocket，PC或Mac版小飞机，

 ## 说明
 -  `/ DIST`位节点生成的目录，这个目录是给[ netlify ]（https://app.netlify.com/）应用的
 -  `node.txt`为SS或SSR节点列表文件（支持类的bash脚本注释，举个栗子：`＃`注释）
 -  `app.js`为项目的核心逻辑
